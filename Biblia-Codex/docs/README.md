## A Palavra de Deus ao Seu Alcance, Onde Quer que Você Esteja.

A Bíblia Codex é um aplicativo bíblico moderno e responsivo, projetado para oferecer uma experiência de estudo e leitura imersiva e personalizada em dispositivos Android e tablets. Com uma interface intuitiva e recursos avançados, a Bíblia Codex busca facilitar o acesso à Palavra de Deus, promovendo a reflexão e o aprofundamento teológico.

Este projeto é um esforço contínuo para integrar tecnologia e fé, proporcionando uma ferramenta poderosa para estudantes da Bíblia, teólogos e qualquer pessoa que deseje explorar as Escrituras de forma mais dinâmica e interativa.

## Funcionalidades Principais

A Bíblia Codex oferece uma gama de funcionalidades pensadas para enriquecer sua jornada espiritual e acadêmica:

*   **Leitura Adaptativa**: Layout otimizado para diferentes tamanhos de tela, garantindo uma experiência de leitura confortável em smartphones e tablets.
*   **Modos de Leitura Personalizáveis**: Escolha entre modos claro, sépia, escuro e preto para adaptar a visualização às suas preferências e condições de iluminação.
*   **Tipografia Flexível**: Ajuste o tamanho da fonte, altura da linha e espaçamento entre letras, além de selecionar entre diversas famílias de fontes (Serif, Sans Serif, Monospace) para uma leitura ideal.
*   **Ferramentas de Estudo Integradas**: Acesse dicionários teológicos, comentários bíblicos e referências cruzadas para aprofundar seu entendimento dos textos.
*   **Notas e Destaques**: Crie anotações pessoais e destaque versículos importantes para organizar seus estudos.
*   **Planos de Leitura**: Acompanhe planos de leitura personalizados para manter a consistência em sua jornada bíblica.
*   **Sincronização em Nuvem**: Mantenha seus dados e configurações sincronizados em todos os seus dispositivos.
*   **Suporte a Módulos MySword/MyBible**: Importe e utilize módulos de estudo externos para expandir sua biblioteca de recursos.
*   **Assistente de Estudo com IA**: Utilize a inteligência artificial para obter insights e análises teológicas sobre passagens bíblicas.

## Tecnologias Utilizadas

O desenvolvimento da Bíblia Codex é impulsionado por uma robusta pilha de tecnologias modernas, garantindo performance, escalabilidade e uma experiência de usuário fluida:

| Categoria         | Tecnologia/Biblioteca                                   | Descrição                                                                                             |
| :---------------- | :------------------------------------------------------ | :---------------------------------------------------------------------------------------------------- |
| **Frontend**      | React.js, Vite, TypeScript, TailwindCSS                 | Interface de usuário reativa, rápido ambiente de desenvolvimento, tipagem segura e estilização utilitária. |
| **Mobile**        | Capacitor.js, Android                                   | Empacotamento da aplicação web para plataformas nativas Android, com acesso a recursos do dispositivo. |
| **Estado/Dados**  | IndexedDB (idb), Firebase                               | Armazenamento local otimizado e serviços de backend para autenticação e sincronização em nuvem.       |
| **IA**            | Google GenAI                                            | Integração com modelos de inteligência artificial para funcionalidades de estudo e análise.           |
| **Utilidades**    | clsx, date-fns, docx, dotenv, express, file-saver, lucide-react, motion, react-markdown, sql.js, tailwind-merge | Diversas bibliotecas para manipulação de classes CSS, datas, documentos, variáveis de ambiente, ícones, animações, renderização Markdown e SQLite no navegador. |

## Créditos Open Source

A Bíblia Codex é construída sobre os ombros de gigantes do software livre. Agradecemos imensamente a todos os projetos e comunidades open-source que tornam este aplicativo possível:

*   **React**: Uma biblioteca JavaScript para construir interfaces de usuário.
*   **Vite**: Um ambiente de desenvolvimento frontend de próxima geração.
*   **TypeScript**: Um superset tipado de JavaScript que compila para JavaScript puro.
*   **Tailwind CSS**: Um framework CSS utilitário para construção rápida de interfaces.
*   **Capacitor**: Um framework de código aberto para construir aplicativos web nativos.
*   **Firebase**: Uma plataforma de desenvolvimento de aplicativos móveis e web do Google.
*   **Google GenAI**: Ferramentas e APIs para integrar recursos de IA generativa.
*   **sql.js**: Uma porta SQLite para JavaScript, permitindo o uso de bancos de dados SQLite no navegador.
*   **Lucide React**: Uma biblioteca de ícones simples e personalizáveis.
*   **Framer Motion**: Uma biblioteca de animação para React.
*   **date-fns**: Uma biblioteca de utilitários de data moderna.
*   E muitas outras bibliotecas e ferramentas que compõem nosso ecossistema de desenvolvimento.

## Como Contribuir

Se você é um desenvolvedor, designer ou entusiasta e deseja contribuir com o projeto Bíblia Codex, sinta-se à vontade para:

1.  Fazer um fork do repositório.
2.  Criar uma nova branch para sua funcionalidade (`git checkout -b feature/sua-funcionalidade`).
3.  Realizar suas alterações e testá-las.
4.  Enviar um Pull Request detalhado com suas modificações.

Todas as contribuições são bem-vindas e apreciadas!
